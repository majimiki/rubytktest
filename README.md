2019年ごろに作ったRuby/TKアプリ
javascriptで作り直したい

機能
・指定条件でファイルを検索して登録。（ベースディレクトリ、拡張子、含むワード、無視ワード）
・拡張子別に指定したプログラムに渡して実行。
・実行時間、実行回数を記録。
・ランダムソート、ランダム実行。
・ファイルの移動、リネームを自動検出して実行時間回数などのデータも移行
